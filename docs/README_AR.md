# 📊 لوحة تحكم سيرفر FiveM

لوحة تحكم احترافية وآمنة لإدارة سيرفر FiveM من أي مكان عن بعد!

## المميزات الرئيسية ✨

### 🎮 لوحة المعلومات
- عرض عدد اللاعبين الحاليين
- معلومات وقت التشغيل
- إحصائيات الأداء (FPS، الذاكرة، CPU)
- حالة السيرفر الحية

### 👥 إدارة اللاعبين
- قائمة كاملة بجميع اللاعبين
- معلومات اللاعب (المستوى، وقت اللعب)
- طرد اللاعبين (Kick)
- حظر اللاعبين (Ban)
- البحث والتصفية

### ⚙️ إعدادات السيرفر
- إعادة تشغيل السيرفر
- إيقاف/تشغيل السيرفر
- مراقبة الموارد
- الإحصائيات التفصيلية

### 📦 إدارة الموارد
- عرض جميع الموارد المثبتة
- تشغيل الموارد
- إيقاف الموارد
- مراقبة حالة الموارد

### 📋 السجلات
- سجل النشاطات الكامل
- رصد الأخطاء
- تاريخ التعديلات

## التثبيت 🚀

### المتطلبات
- Node.js (v14 أو أعلى)
- npm أو yarn
- سيرفر FiveM

### خطوات التثبيت

1. **استنساخ المستودع**
```bash
git clone https://github.com/asf517711-design/fivem-dashboard.git
cd fivem-dashboard
```

2. **تثبيت الحزم**
```bash
npm install
```

3. **إعداد متغيرات البيئة**
```bash
cp .env.example .env
# عدّل .env حسب احتياجاتك
```

4. **تشغيل السيرفر**
```bash
npm start
```

5. **فتح اللوحة**
```
http://localhost:5000
```

## API Endpoints 🔌

### المصادقة
- `POST /api/auth/login` - تسجيل الدخول
- `GET /api/auth/profile` - معلومات المستخدم

### اللاعبون
- `GET /api/players` - قائمة اللاعبين
- `GET /api/players/:id` - معلومات لاعب
- `POST /api/players/:id/kick` - طرد لاعب
- `POST /api/players/:id/ban` - حظر لاعب

### السيرفر
- `GET /api/server/info` - معلومات السيرفر
- `GET /api/server/stats` - الإحصائيات
- `POST /api/server/restart` - إعادة التشغيل
- `POST /api/server/stop` - الإيقاف
- `GET /api/server/resources` - الموارد
- `POST /api/server/resources/:name/start` - تشغيل مورد
- `POST /api/server/resources/:name/stop` - إيقاف مورد
