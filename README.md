# FiveM Server Dashboard

لوحة تحكم سيرفر فايف ام عن بعد احترافية وآمنة

## المميزات ✨

- 📊 عرض إحصائيات السيرفر
- 👥 إدارة اللاعبين
- 🔐 نظام أمان قوي
- 🎨 واجهة حديثة وسهلة
- 📱 متوافقة مع الهاتف
- 🌙 واجهة عربية

## التثبيت

```bash
git clone https://github.com/asf517711-design/fivem-dashboard.git
cd fivem-dashboard
npm install
```

## البدء

```bash
npm start
```

## الهيكل

```
fivem-dashboard/
├── server/          # Backend API
├── client/          # Frontend UI
├── resources/       # FiveM Resources
```
